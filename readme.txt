Enables site owners to implement the ILS exchange rate on their WP sites.

== Description ==

ILS Exchange Rate is an extremely simple plugin for Israeli WordPress users.
Implement an exchange rate table on your site by using the shortcode [ILSES] wherever you want the table to appear.

This plugin uses the official "Bank of Israel"'s XML feed which updates daily. You can access the XML by going to: "http://www.boi.org.il/currency.xml".
You can read the terms of use here: https://www.boi.org.il/he/Pages/disclaimer.aspx
